WYSMarkdown Type to generate Markdown

Authors Notes:

This editor and Markdown converter started as a side projec to help create quick markdown documents.

While Markdown can do a lot more than the few features shown in the editor, I hope that this helps for Markdown starters to get going.

Please feel free to form and expand this editor on my Github account.
